# TSV-to-EXCEL-Converter
### An easy to use CLI based file converter from tsv to xlsx

- Do this [✅]
``` python main.py -i name.tsv ```
or 
``` python main.py -in name.tsv ```

- Not this [❌]
``` python main.py ```
or 
``` python main.py -i somethingelsefile.abc ```
